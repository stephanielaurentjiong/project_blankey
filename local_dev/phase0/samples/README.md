# Sample Data

This directory contains sample images and descriptions for testing the caption generator.

## Files

- `video_cover1.jpg` - Sample video cover image
- `sample_descriptions.json` - Test cases with descriptions and expected styles

## Usage

The sample descriptions JSON contains test cases that can be used for:
- Local testing
- Validation
- Performance benchmarking

Each sample includes:
- `id`: Unique identifier
- `image`: Image filename
- `description`: Video description text
